# Crazy Trout Arena — CRM Prototype

Интерактивный прототип CRM-системы для платного пруда премиум-класса.

## 🚀 GitHub Pages

**https://smartmoneymoscow-cell.github.io/CRM-/**

## 📱 Режимы просмотра

| Режим | Описание |
|-------|----------|
| 👤 Клиент · Десктоп | Веб-кабинет клиента с sidebar |
| 👤 Клиент · Мобайл | Мобильное приложение клиента (phone frame) |
| ⚙️ Админ · Десктоп | Веб-панель администратора |
| ⚙️ Админ · Мобайл | Мобильное приложение администратора |

## 📱 Экраны клиента

| # | Экран | Описание |
|---|-------|----------|
| 1 | Регистрация | Имя, телефон, пароль с индикатором |
| 2 | Вход | Телефон + пароль |
| 3 | Восстановление | Ввод телефона для сброса |
| 4 | Новый пароль | OTP-код + новый пароль |
| 5 | Карта пруда | Интерактивная карта с секторами |
| 6 | Бронирование | Выбор сектора, тарифа, оплата |
| 7 | Профиль | Статистика, лояльность, меню |
| 8 | Рейтинги | Таблица лидеров |
| 9 | Бонусы | Баланс, призы, история |

## ⚙️ Экраны админа

| # | Экран | Описание |
|---|-------|----------|
| 1 | Карта и чек | Карта пруда + создание чека |
| 2 | Новый чек | Выбор клиента, сектора, тарифа |
| 3 | Лента чеков | Список чеков с фильтрами |
| 4 | Финансы | Выручка, расходы, графики |
| 5 | Расходы | Управление расходами |
| 6 | Пользователи | Список клиентов |
| 7 | Статистика улова | По видам рыбы, графики |
| 8 | Профиль админа | Статистика, навигация |

## 🎨 Дизайн-система

Все токены из оригинального Pitch Deck:
- `--orange: #E89829` · `--ink: #14130F` · `--paper: #FAF6EE`
- `--teal: #2A6A7E` · `--green: #3FA66B` · `--gold: #D4A85A`
- Шрифты: Inter + Inter Tight (Google Fonts)
