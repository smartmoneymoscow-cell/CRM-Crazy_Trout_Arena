# Crazy Trout Arena — CRM Client Prototype

Интерактивный муляж клиентского веб-интерфейса для CRM-системы бронирования рыбалки.

## 🚀 GitHub Pages

Откройте: `https://smartmoneymoscow-cell.github.io/CRM-/`

## 📱 Экраны

| # | Экран | Описание |
|---|-------|----------|
| 1 | Заставка | Анимированный splash screen |
| 2 | Авторизация | Ввод номера телефона |
| 3 | OTP-код | Подтверждение через SMS |
| 4 | Главная | Карта, поиск, рекомендации |
| 5 | Водоёмы | Карта + список водоёмов |
| 6 | Детали водоёма | Описание, отзывы, цены |
| 7 | Бронирование | Выбор даты, смены, гостей |
| 8 | Оплата | Выбор способа оплаты |
| 9 | Подтверждение | Номер брони, детали |
| 10 | Профиль | Статистика, лояльность |
| 11 | История | Список бронирований |
| 12 | Статистика | Графики улова |

## 🎨 Дизайн-система

- **Шрифты:** Inter + Inter Tight (Google Fonts)
- **Палитра:** `--orange: #E89829`, `--ink: #14130F`, `--paper: #FAF6EE`
- **Радиусы:** 9px / 12px / 14px
- **Масштаб:** Mobile-first, 375px viewport

## 🛠 Стек

Чистый HTML + CSS + Vanilla JS. Без зависимостей.

## 📂 Структура

```
├── index.html          # SPA — все экраны
├── css/style.css       # Стили + дизайн-токены
├── js/app.js           # Навигация + интерактивность
└── README.md
```

## 💻 Локальный запуск

```bash
# Просто откройте index.html в браузере
open index.html

# Или через локальный сервер
python3 -m http.server 8000
# → http://localhost:8000
```
